# ListGroup

`ListGroup` displays a list of elements.

<Preview>
  <Story name="listgroup">
    <ListGroup padded>
      <ListItem>
        <div>here is a list</div>
      </ListItem>
      <ListItem>
        <div>of some items</div>
      </ListItem>
      <ListItem>
        <div>in a list group</div>
      </ListItem>
    </ListGroup>
  </Story>
</Preview>

A `ListGroup` may be sized `sm`, `md`, or `lg`.
