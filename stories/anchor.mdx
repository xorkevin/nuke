# Anchor

`Anchor` is used to create links.

<Preview>
  <Story name="anchor">
    <Anchor>This is an unstyled link.</Anchor>
  </Story>
</Preview>

The `ext` boolean prop may be used for external links.

<Preview>
  <Story name="anchor ext">
    <Anchor ext href="https://xorkevin.com">https://xorkevin.com</Anchor>
  </Story>
</Preview>

The variant `anchor/text` is used to create styled links for text.

<Preview>
  <Story name="anchor text">
    <div>This is a link for <AnchorText>some text</AnchorText>.</div>
  </Story>
</Preview>

The variant `anchor/secondary` is used to create a more subdued text link.

<Preview>
  <Story name="anchor secondary">
    <div>This is a link for <AnchorSecondary>some text</AnchorSecondary>.</div>
  </Story>
</Preview>

The variant `anchor/nocolor` is used to create a text link that inherits its
color.

<Preview>
  <Story name="anchor nocolor">
    <div>This is a link for <AnchorNoColor>some text</AnchorNoColor>.</div>
  </Story>
</Preview>
